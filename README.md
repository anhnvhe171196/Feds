# FPT Electronic Device Shop (FEDS)

## Overview
An e-commerce web application for selling electronic devices such as laptops and phones.

## Features
- User authentication and role-based authorization
- Shopping cart and checkout functionality
- VNPay payment integration
- Product, user, and dashboard management (admin)

## Tech Stack
- Java (JSP/Servlet)
- SQL Server
- HTML, CSS, JavaScript, Bootstrap
